[FR]
Nom du projet : bigint
Fichiers attendus : bigint.hpp, bigint.cpp
--------------------------------------------------------------------------------
En informatique, un bignum est un objet représentant un nombre de précision
arbitraire, ce qui est utile quand vous voulez stocker un nombre plus grand que
SIZE_MAX sans aucune perte de précision. Ceci est généralement réalisé en
stockant un tableau ou une chaîne de caractères contenant les différentes
« parties » du nombre.
Créez une classe appelée bigint qui stockera un entier non signé de précision
arbitraire.
Votre classe doit supporter l'addition, la comparaison et le « décalage de
chiffres » (comme le décalage de bits mais au lieu de décaler les bits vous
décalerez les chiffres en base 10, par exemple : (42 << 3 == 42000) et 
(1337 >> 2 == 13)).
Votre bigint doit être affichable avec l'opérateur << (en base 10) et la sortie
ne doit pas contenir de zéros non significatifs.
Vous trouverez un main dans ce répertoire qui doit fonctionner avec votre classe

 
[MAIN DU SUJET]
#include "bigint.hpp"
#include <iostream>

int main()
{
    const bigint a(42);
    bigint b(21), c, d(1337), e(d);

    std::cout << "a = " << a << std::endl;
    std::cout << "b = " << b << std::endl;
    std::cout << "c = " << c << std::endl;
    std::cout << "d = " << d << std::endl;
    std::cout << "e = " << e << std::endl;

    std::cout << "a + b = " << a + b << std::endl;
    std::cout << "(c += a) = " << (c += a) << std::endl;

    std::cout << "b = " << b << std::endl;
    std::cout << "++b = " << ++b << std::endl;
    std::cout << "b++ = " << b++ << std::endl;

    std::cout << "(b << 10) + 42 = " << ((b << 10) + 42) << std::endl;
    std::cout << "(d <<= 4) = " << (d <<= 4) << std::endl;
    std::cout << "(d >>= 2) = " << (d >>= (const bigint)2) << std::endl;

    std::cout << "a =" << a << std::endl;
    std::cout << "d =" << d << std::endl;

    std::cout << "(d < a) = " << (d < a) << std::endl;
    std::cout << "(d <= a) = " << (d <= a) << std::endl;
    std::cout << "(d > a) = " << (d > a) << std::endl;
    std::cout << "(d >= a) = " << (d >= a) << std::endl;
    std::cout << "(d == a) = " << (d == a) << std::endl;
    std::cout << "(d != a) = " << (d != a) << std::endl;
}

number, this is useful when you want to store a number bigger than SIZE_MAX
without any loss of precision. This is often achieved by storing an array or a
string containing the different "parts" of the number.
Create a class called bigint that will store an arbitrary precision unsigned
integer.

instead of shifting the bits you will shift the digits in base 10, e.g.:
(42 << 3 == 42000) and (1337 >> 2 == 13)).
Your bigint must be printable with the << operator (in base 10) and the output
should not contain any leading zeros.
